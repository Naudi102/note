## 创建目录

在 Ubuntu 中，可以通过图形界面或命令行创建目录（文件夹）。

### 图形界面方式
- 右键 → “新建文件夹”（New Folder）。
- **限制**：一次只能创建一层目录。
- **权限要求**：当前路径必须有写权限，否则“新建文件夹”按钮为灰色。

*![](http://localhost:8000/static/screenshots/screenshot_01c307628-c80c-4cb3-8b90-1db8c79fbb57.jpg)*

### 命令行方式（使用 `mkdir`）
- 基本语法：
  ```bash
  mkdir 目录名
  ```
- 示例：
  ```bash
  mkdir 修计
  ```
  创建名为“修计”的目录。

- **递归创建多层目录**：使用 `-p` 参数（视频中称为“杠屁”，即 `-p`）。
  ```bash
  mkdir -p 修计/取经/图地
  ```
  即使中间目录（如“取经”）不存在，也能一次性创建完整路径。

> 若不加 `-p`，而父目录不存在，系统会报错：“No such file or directory”。

*![](http://localhost:8000/static/screenshots/screenshot_199e56dd8-4682-4eda-8870-95f5633517d1.jpg)*

## 创建文件

### 命令行方式（使用 `touch`）
- 基本语法：
  ```bash
  touch 文件名
  ```
- 示例：
  ```bash
  touch 孙悟空.txt
  ```
  在当前目录下创建一个名为“孙悟空.txt”的**空文件**。

- 可指定路径：
  ```bash
  touch 修计/取经/图地/孙悟空.txt
  ```
  （前提是路径已存在，否则需先用 `mkdir -p` 创建）

### 图形界面方式
- Ubuntu 图形界面**没有直接“新建文件”选项**。
- 需通过文本编辑器（如 **gedit**，视频中称“文本文档”）创建：
  1. 打开 gedit。
  2. 编辑内容（可为空）。
  3. 点击“保存” → 选择路径（如桌面/修计/取经/图地）→ 输入文件名（如“齐天大圣.txt”）。
  4. 即使内容为空，也会创建文件。

> 注意：不输入扩展名（如 `.txt`）也可以，但建议加上以便识别文件类型。

## 复制文件与目录（cp 命令）

`cp`（copy）用于复制文件或目录。

### 基本用法
- 复制文件：
  ```bash
  cp 源文件 目标路径
  ```

- **递归复制整个目录**：使用 `-r` 参数（视频中称“杠 R”）。
  ```bash
  cp -r 取经 ~/Desktop/
  ```
  将“取经”目录及其所有内容复制到桌面。

> 路径说明：
> - `~/Desktop/` 表示当前用户的桌面路径。
> - `~` 是家目录的简写。

## 安装中文输入法（搜狗输入法）

由于默认 Ubuntu 系统未安装中文输入法，输入中文不便。推荐安装 **搜狗输入法（Sogou Input Method）**。

### 安装步骤
1. 打开浏览器，访问 [百度](https://www.baidu.com)。
2. 搜索“搜狗输入法 Linux 版”。
3. 进入官网，下载对应架构版本（如 x86_64，适用于 Intel/AMD 芯片）。
4. 下载 `.deb` 安装包后，双击通过图形界面安装。
5. 安装过程中需输入用户密码进行认证。
6. 安装完成后，在系统设置 → 区域与语言 → 输入源中，添加“**中文 (智能拼音)**”。
7. 切换输入法快捷键通常为 `Super + 空格` 或 `Ctrl + Space`。

### 使用效果
- 可在 gedit、终端等应用中直接输入中文（如“齐天大圣”）。
- 支持全角/半角标点、简体中文等。

> 提示：若系统资源较低，安装过程可能稍慢。

## AI 总结

本节详细介绍了在 Ubuntu 系统中创建目录与文件的多种方法。通过 `mkdir -p` 可高效创建多级目录，`touch` 命令快速生成空文件，而图形界面则依赖文本编辑器间接创建文件。此外，`cp -r` 支持递归复制整个目录结构。为提升中文输入体验，推荐安装搜狗输入法，并通过系统设置启用。这些基础操作是 Linux 系统日常使用和开发工作的核心技能。
