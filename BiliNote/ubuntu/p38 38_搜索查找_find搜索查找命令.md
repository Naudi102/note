## find 命令概述

`find` 是 Linux/Ubuntu 系统中用于**搜索文件和目录**的强力命令，支持多种搜索条件：

- 按**文件名**（最常用）
- 按**文件所有者（用户）**
- 按**文件大小**

基本语法：
```bash
find [搜索路径] [选项] [参数]
```

## 按文件名搜索（最常用）

### 基本用法
在指定目录下精确查找文件名：
```bash
find . -name "redis.txt"
```
- `.` 表示当前目录
- `-name` 指定按名称查找
- 文件名需用引号包裹（尤其含特殊字符时）

> *![](http://localhost:8000/static/screenshots/screenshot_0c118bf38-1968-4899-a2e8-f323a323231f.jpg)*：演示在 `OBD` 目录下成功查找到 `redis.txt` 的输出结果。

### 模糊搜索（通配符）
当不记得完整文件名时，可使用通配符进行模糊匹配：

- `*`：匹配任意数量的任意字符
- `.`：在文件扩展名中表示字面意义的点（如 `.txt`）

示例：查找所有 `.txt` 结尾的文件
```bash
find . -name "*.txt"
```

> **注意**：`*` 和 `.` 需配合使用，`*.txt` 表示“任意前缀 + .txt 后缀”。

> *![](http://localhost:8000/static/screenshots/screenshot_1f90ce109-c00d-403a-b48f-f2bf93ef46da.jpg)*：展示 `find . -name "*.txt"` 返回多个 `.txt` 文件的结果。

## 按所有者（用户）搜索

使用 `-user` 选项查找属于特定用户的文件：
```bash
find . -user zhangzong
```
- `zhangzong` 是系统中存在的用户名
- 会列出该用户拥有的**所有文件和目录**

> 此功能适用于权限管理或排查用户生成的文件。

> *![](http://localhost:8000/static/screenshots/screenshot_2bdc28bd1-3f34-4d01-9f20-4de611871d89.jpg)*：执行 `find . -user zhangzong` 成功返回属于该用户的所有文件。

## 按文件大小搜索

使用 `-size` 选项按文件大小筛选：

### 大小单位
- `c`：字节（bytes）
- `k`：千字节（KB）
- `M`：兆字节（MB）
- `G`：吉字节（GB）

### 比较符号
- `+`：大于（如 `+200k` 表示大于 200KB）
- `-`：小于（如 `-1M` 表示小于 1MB）
- 无符号：等于（极少使用，因精确匹配难）

示例：
```bash
find . -size +200k    # 查找大于 200KB 的文件
find . -size +1M      # 查找大于 1MB 的文件
```

> **提示**：值越大，匹配结果越少。适合用于清理大文件或排查磁盘占用。

## AI 总结

`find` 命令是 Linux 系统中功能强大且常用的文件搜索工具。最核心的用法是通过 `-name` 进行文件名搜索，支持通配符 `*` 实现模糊匹配；此外，还可通过 `-user` 按所有者查找，或通过 `-size` 按文件大小筛选（常用 `+` 表示“大于”）。掌握这些基础用法，能极大提升在终端中定位文件的效率。建议优先熟练使用按名称搜索，其他功能按需调用。
