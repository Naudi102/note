## for 循环概述

在 Shell 脚本中，`for` 循环是流程控制的重要组成部分，用于重复执行一段代码。视频中介绍了 **两种 `for` 循环的语法**，分别适用于不同的场景：

1. **类 C 语法**：适用于需要控制起始值、终止条件和步长的数值循环（如从 1 加到 100）。
2. **in 列表语法**：适用于遍历一组已知元素（如字符串、文件名等）。

---

## 第一种语法：C 风格 for 循环（类 C 语法）

### 语法结构
```bash
for (( 初始值; 循环条件; 变量变化 ))
do
    # 循环体
done
```

### 示例：计算 1 到 100 的和
```bash
#!/bin/bash

sum=0
for (( i=1; i<=100; i++ ))
do
    sum=$((sum + i))
done

echo $sum
```

### 关键点说明
- **初始值**：`i=1`，定义循环变量的起始值。
- **循环条件**：`i<=100`，只要条件为真，循环继续。
- **变量变化**：`i++` 等价于 `i=$((i + 1))`，每次循环后变量递增。
- **算术运算**：必须使用 `$((...))` 语法进行数学计算，例如 `sum=$((sum + i))`。
- **输出结果**：使用 `echo $sum` 打印最终累加结果（应为 5050）。

（显示终端运行脚本后输出 5050）

### 注意事项
- 分号 `;` 是语法必需的，用于分隔三个部分。
- 不支持直接使用 `<`, `>` 等符号进行比较（但在 `((...))` 中支持）。
- 此语法仅在 `bash` 中有效，非 POSIX 标准。

## 第二种语法：遍历列表的 for 循环（in 列表语法）

### 语法结构
```bash
for 变量 in 值1 值2 值3 ...
do
    # 循环体
done
```

### 示例：遍历老师姓名并输出
```bash
#!/bin/bash

for i in 蔡老师 虚老师 坤老师
do
    echo "班长爱着 $i"
done
```

### 输出结果
```
班长爱着 蔡老师
班长爱着 虚老师
班长爱着 坤老师
```

### 关键点说明
- **列表元素**：由空格分隔，可以是字符串、数字、文件名等。
- **变量引用**：在循环体内通过 `$i` 引用当前元素。
- **灵活性高**：适合处理已知的一组离散值，无需手动控制索引。

（显示终端运行脚本后输出三行“班长爱着 X老师”）

### 注意事项
- 元素之间必须用**空格**分隔，不能用逗号。
- 如果元素包含空格（如 `"张 三"`），需用引号包裹。
- 若省略 `in ...` 部分，默认遍历脚本的位置参数（`$@`）。

## AI 总结

本节详细讲解了 Shell 脚本中 `for` 循环的两种核心语法：  
1. **类 C 语法**（`for ((...))`）适用于需要精确控制循环次数和步长的数值计算场景，如累加求和；  
2. **in 列表语法**（`for var in ...`）适用于遍历预定义的元素集合，如字符串列表或文件名。  

关键实践要点包括：使用 `$((...))` 进行算术运算、注意空格分隔列表元素、以及理解两种语法的适用边界。掌握这两种形式，可高效应对大多数 Shell 自动化任务中的循环需求。
