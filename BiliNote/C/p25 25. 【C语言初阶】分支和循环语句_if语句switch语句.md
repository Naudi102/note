## C语言中的语句分类

C语言中的语句分为以下五类：
- **表达式语句**：如 `a = b + c;`
- **函数调用语句**：如 `printf("Hello");`
- **控制语句**：用于控制程序流程（本节重点）
- **复合语句（代码块）**：由 `{}` 包围的一组语句
- **空语句**：仅包含一个分号 `;`

> 本节课重点讲解的 **if** 和 **switch** 语句，以及后续的 **while/do-while/for** 循环语句，均属于**控制语句**。

## 结构化程序设计

C语言是一种**结构化程序设计语言**，其核心思想是：任何程序逻辑均可由以下三种基本结构组合而成：

1. **顺序结构**：从上到下依次执行，无跳转。
2. **选择结构（分支结构）**：根据条件决定执行哪条路径（如 if、switch）。
3. **循环结构**：重复执行某段代码（如 while、for）。

> 程序员需具备将现实问题抽象为这三种结构的能力，而 C 语言通过控制语句提供了实现这些结构的语法支持。

## 控制语句概述

C语言共有 **9 种控制语句**，可分为三类：

- **条件判断语句（分支语句）**：
  - `if`
  - `switch`
- **循环语句**：
  - `while`
  - `do-while`
  - `for`
- **转向语句**：
  - `break`
  - `continue`
  - `goto`
  - `return`

这些语句共同用于**控制程序的执行流程**，实现各种程序结构。

## if 语句详解

### 基本语法与执行逻辑

```c
if (表达式) {
    // 表达式为真（非零）时执行
}
```

- 表达式结果为 **非零（true）** 时，执行后续语句；为 **0（false）** 时不执行。
- 注意：C语言中 **非零即为真**，0 为假。

### 单分支、双分支与多分支

- **单分支**：只有 `if`
- **双分支**：`if...else`
- **多分支**：`if...else if...else if...else`

示例：根据年龄判断人生阶段
```c
int age = 25;
if (age < 18) {
    printf("青少年\n");
} else if (age >= 18 && age < 28) {
    printf("青年\n");
} else if (age >= 28 && age < 40) {
    printf("中年\n");
} else if (age >= 40 && age < 60) {
    printf("壮年\n");
} else if (age >= 60 && age <= 100) {
    printf("老年\n");
} else {
    printf("老寿星\n");
}
```

> ⚠️ **重要**：多分支中**只会执行其中一个分支**，一旦某个条件满足，其余分支将被跳过。

### 代码块与大括号的使用

- `if` 或 `else` 后若需执行**多条语句**，**必须用 `{}` 包裹**形成代码块。
- 即使只有一条语句，也**强烈建议使用 `{}`**，以避免逻辑错误和提高可读性。

错误示例：
```c
if (age < 18)
    printf("未成年\n");
    printf("不能饮酒\n"); // 此行不受 if 控制！
```

正确写法：
```c
if (age < 18) {
    printf("未成年\n");
    printf("不能饮酒\n");
}
```

### 常见错误与注意事项

1. **悬空 else 问题**：
   ```c
   if (a == 0)
       if (b == 2)
           printf("A=0, B=2\n");
   else
       printf("B!=2\n"); // 此 else 实际匹配内层 if！
   ```
   - **规则**：`else` 总是与**最近的、未配对的 `if`** 配对。
   - **解决方法**：使用 `{}` 明确作用范围。

2. **赋值 vs 比较**：
   - `if (a = 5)` 是赋值，结果为 5（真），常导致逻辑错误。
   - 应写为 `if (a == 5)`。
   - **防御性技巧**：将常量写在左边，如 `if (5 == a)`。若误写为 `if (5 = a)`，编译器会报错。

3. **区间判断**：
   - 错误写法：`if (18 < age < 28)` —— 实际等价于 `if ((18 < age) < 28)`，因 `(18 < age)` 结果为 0 或 1，恒小于 28。
   - 正确写法：`if (age >= 18 && age < 28)`

## switch 语句详解

### 基本语法与执行流程

```c
switch (整型表达式) {
    case 常量1:
        // 语句
        break;
    case 常量2:
        // 语句
        break;
    default:
        // 默认语句
}
```

- 根据**整型表达式**的值，跳转到匹配的 `case` 标签处开始执行。
- **执行特点**：从匹配的 `case` 开始，**顺序向下执行**，直到遇到 `break` 或 `switch` 结束。

### break 的作用与贯穿（fall-through）

- `break` 用于**跳出当前 `switch` 语句**。
- 若省略 `break`，程序会继续执行后续 `case` 的代码（称为“贯穿”或 “fall-through”）。

示例：利用贯穿特性简化代码
```c
int day = 3;
switch (day) {
    case 1:
    case 2:
    case 3:
    case 4:
    case 5:
        printf("工作日\n");
        break;
    case 6:
    case 7:
        printf("周末\n");
        break;
}
```

> ✅ 贯穿是合法且有用的特性，但需**谨慎使用**，避免意外逻辑错误。

### default 分支

- 当所有 `case` 都不匹配时，执行 `default` 分支。
- `default` 可放在 `switch` 中任意位置，但**通常置于末尾**。
- 即使 `default` 在最后，也**建议加上 `break`**，以防未来新增 `case` 时出错。

### 使用限制与注意事项

1. **表达式类型限制**：
   - `switch` 后的表达式必须是**整型**（包括 `char`，因其 ASCII 码为整数）。
   - **不支持**浮点型（`float`, `double`）、字符串等。

2. **case 标签要求**：
   - 必须是**整型常量表达式**（如 `1`, `'a'`, `2+3`）。
   - **不能是变量**（如 `case n:` 非法）。

3. **嵌套 switch**：
   - `switch` 可嵌套使用。
   - 内层 `break` **只跳出当前层**的 `switch`。

## 代码风格与编程习惯

良好的代码风格能显著提升可读性、可维护性，并减少 Bug。

### 变量命名规范

- 避免使用无意义的名称（如 `a`, `b`）。
- 使用**有意义的英文单词**，如 `age`, `studentCount`。
- 推荐命名风格：
  - **下划线命名法**：`first_name`
  - **驼峰命名法**：`firstName`

### 空格与缩进

- 运算符两侧加空格：`int a = 10;` 优于 `inta=10;`
- 使用一致的缩进（通常 4 空格或 1 个 Tab）。
- VS 等现代 IDE 可自动格式化代码。

### 防御性编程技巧

- **常量前置比较**：`if (5 == a)` 而非 `if (a == 5)`，可防止误写为赋值。
- **始终使用代码块**：即使 `if` 后只有一条语句，也用 `{}` 包裹。
- **善用注释和空行**：分隔逻辑块，提高可读性。

> 📚 **推荐书籍**：《高质量C/C++编程指南》（林锐博士著）——系统讲解编码规范。

## 练习与示例

### 示例1：判断奇数
```c
int num;
scanf("%d", &num);
if (num % 2 == 1) {
    printf("是奇数\n");
} else {
    printf("不是奇数\n");
}
```

### 示例2：输出1-100所有奇数
**方法一**：遍历 + 判断
```c
for (int i = 1; i <= 100; i++) {
    if (i % 2 == 1) {
        printf("%d ", i);
    }
}
```

**方法二**：直接生成奇数（更高效）
```c
for (int i = 1; i <= 100; i += 2) {
    printf("%d ", i);
}
```

### 示例3：switch 执行流程分析
```c
int n = 1, m = 2;
switch (n) {
    case 1:
        n++; // n=2
    case 2:
        n++; // n=3
        switch (n) { // n=3
            case 2:
                n++;
            case 3:
                n++; // n=4
                break; // 跳出内层 switch
        }
    case 3:
        n++; // n=5
        break;
}
// 最终 n=5, m=2
```

> 🔍 关键点：内层 `break` 不影响外层 `switch` 的贯穿行为。

## AI 总结

本节深入讲解了 C 语言中的分支控制语句 `if` 和 `switch`。核心要点包括：`if` 语句通过布尔表达式实现单/双/多分支逻辑，需注意代码块的使用以避免悬空 `else` 等陷阱；`switch` 语句适用于多路整型分支，其独特的“贯穿”特性既可简化代码，也易引发 Bug，需配合 `break` 谨慎使用。此外，课程强调了**代码风格**的重要性——良好的命名、缩进和防御性编程习惯是写出健壮、可维护代码的基础。掌握这些知识，是构建复杂程序逻辑的关键第一步。
