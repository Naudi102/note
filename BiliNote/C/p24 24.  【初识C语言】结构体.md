## 指针声明风格的讨论

在 C 语言中，指针声明时 `*` 的位置存在不同风格，这属于 **代码风格（Coding Style）** 问题，无绝对对错，但需理解其含义：

- **VS（Visual Studio）风格**：`int* p;`  
  强调 `int*` 是一个整体类型（“指向 int 的指针”），适合单变量声明。
  
- **K&R（Kernighan & Ritchie）风格**：`int *p;`  
  强调 `*` 与变量名绑定，更清晰地反映 C 语言的解析规则。

### 关键注意事项：
- 在**连续声明多个变量**时，风格差异影响语义：
  ```c
  int* p1, p2, p3;  // 错误认知：p1、p2、p3 都是指针
  ```
  实际上：
  - `p1` 是 `int*`（指针）
  - `p2` 和 `p3` 是 `int`（普通整型）

- 正确写法（若都需为指针）：
  ```c
  int *p1, *p2, *p3;
  ```

> ✅ 建议：根据团队规范选择风格，但务必理解 `*` 在语法上属于变量而非类型。

## 结构体（struct）的基本概念

### 为什么需要结构体？
- C 语言内置类型（如 `int`, `char`, `float`）只能表示**单一数据**。
- 现实世界对象（如“学生”、“书籍”）包含**多个不同类型的数据**，需组合表示。

### 示例：描述一个“学生”
- 姓名（字符串）
- 年龄（整数）
- 性别（字符串）
- 电话（字符串）

→ 内置类型无法直接表达，需自定义复合类型。

### 结构体的定义
- 使用关键字 `struct`
- 将多个**不同类型的变量**组合成一个新类型
- 类似“蓝图”或“设计图”，本身不占用内存

---

## 定义结构体类型

```c
struct STU {
    char name[20];   // 姓名
    int age;         // 年龄
    char sex[10];    // 性别
    char tel[12];    // 电话
};
```

### 说明：
- `struct STU` 是一个**用户自定义类型**
- 大括号 `{}` 内的变量称为 **结构体成员（members）**
- 末尾必须加分号 `;`

> 💡 类比：结构体类型如同“房屋设计图”，不占空间；只有用它创建变量（“盖房子”）才分配内存。

## 创建结构体变量与初始化

### 声明变量
```c
struct STU s;  // s 是 struct STU 类型的变量
```

### 初始化（定义时赋值）
```c
struct STU s = {"张三", 20, "男", "15596668622"};
```

- 使用 `{}` 按**成员顺序**初始化
- 字符串用双引号，整数直接写值
- 成员间用逗号分隔

> ⚠️ 注意：初始化顺序必须与结构体定义中的成员顺序一致。

## 访问结构体成员

### 点操作符（.）

用于**直接通过结构体变量**访问成员：

```c
printf("姓名: %s\n", s.name);
printf("年龄: %d\n", s.age);
printf("性别: %s\n", s.sex);
printf("电话: %s\n", s.tel);
```

- 语法：`变量名.成员名`
- 左侧必须是**结构体变量**（不是指针）

*![](http://localhost:8000/static/screenshots/screenshot_21b3663b5-ac13-4c5e-8034-469f6a8cec26.jpg)*

### 箭头操作符（->）

用于**通过结构体指针**访问成员：

```c
struct STU *ps = &s;  // ps 指向 s

// 方式1：解引用 + 点操作符（繁琐）
printf("%s\n", (*ps).name);

// 方式2：箭头操作符（推荐）
printf("%s\n", ps->name);
```

- 语法：`指针->成员名`
- 等价于 `(*指针).成员名`
- 更简洁，避免冗余括号

### 对比总结
| 操作符 | 左侧类型           | 示例          |
|--------|--------------------|---------------|
| `.`    | 结构体变量         | `s.age`       |
| `->`   | 结构体指针         | `ps->age`     |

> ✅ 规则：有对象用 `.`，有指针用 `->`。

## 结构体使用示例

完整代码演示：

```c
#include <stdio.h>

struct STU {
    char name[20];
    int age;
    char sex[10];
    char tel[12];
};

void print_stu(struct STU *ps) {
    printf("姓名: %s\n", ps->name);
    printf("年龄: %d\n", ps->age);
    printf("性别: %s\n", ps->sex);
    printf("电话: %s\n", ps->tel);
}

int main() {
    struct STU s = {"张三", 20, "男", "15596668622"};
    
    // 直接访问
    printf("方式1:\n");
    printf("姓名: %s\n", s.name);
    
    // 通过指针访问
    printf("方式2:\n");
    print_stu(&s);
    
    return 0;
}
```

输出：
```
方式1:
姓名: 张三
方式2:
姓名: 张三
年龄: 20
性别: 男
电话: 15596668622
```

> ⚠️ 注意：`printf` 格式符必须与数据类型严格匹配，否则可能导致未定义行为（如崩溃）。

## 作业题讲解

### 题目1：计算商和余数
**要求**：输入两个整数 A 和 B（0 ≤ A, B ≤ 10000，B ≠ 0），输出 A / B 的商和余数。

**示例输入**：
```
10 3
```

**示例输出**：
```
3 1
```

**参考代码**：
```c
#include <stdio.h>

int main() {
    int a, b;
    scanf("%d %d", &a, &b);
    printf("%d %d\n", a / b, a % b);
    return 0;
}
```

### 其他作业题要点：
- **关键字（Keywords）**：
  - 不能由用户自定义
  - 不能用作变量名（如 `int`, `struct`, `if` 等）
  - `#define` 不是关键字，而是**预处理指令**

- **switch 语句**：
  - 可包含 `break`、`case`、`default`
  - **不能包含 `continue`**（`continue` 仅用于循环）
