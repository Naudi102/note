## C语言中的循环结构概述

C语言提供了三种循环结构：
- **for 循环**
- **while 循环**
- **do...while 循环**

本节重点讲解 **while 循环**，它是基于条件判断的循环机制，适用于不确定循环次数但有明确终止条件的场景。

---

## while 循环的基本语法与执行流程

### 语法格式
```c
while (表达式) {
    // 循环体（语句）
}
```

### 执行流程
1. 首先计算 `表达式` 的值。
2. 若表达式为 **真（非0）**，则执行循环体。
3. 执行完循环体后，**再次回到步骤1**，重新判断表达式。
4. 若表达式为 **假（0）**，则退出循环，继续执行后续代码。

> ⚠️ 若表达式始终为真，则形成 **死循环（infinite loop）**。

## 示例：使用 while 打印 1 到 10

### 代码实现
```c
#include <stdio.h>

int main() {
    int i = 1;
    while (i <= 10) {
        printf("%d\n", i);
        i++;  // 关键：更新循环变量
    }
    return 0;
}
```

### 执行过程分析
- 初始 `i = 1`
- 每次循环：
  - 打印当前 `i`
  - `i++` 使 `i` 增加 1
- 当 `i = 11` 时，`i <= 10` 为假，循环终止

输出结果：
```
1
2
3
...
10
```

## break 语句在 while 循环中的作用

### 功能
- **立即终止整个 while 循环**
- 跳出循环体，执行循环后的下一条语句

### 示例
```c
int i = 1;
while (i <= 10) {
    if (i == 5) {
        break;  // 当 i 等于 5 时跳出循环
    }
    printf("%d\n", i);
    i++;
}
```

### 输出结果
```
1
2
3
4
```

> 💡 `break` 执行后，**不会执行 `i++`**，也不会再进入循环。

*![](http://localhost:8000/static/screenshots/screenshot_2c680ffd5-a6a1-458c-80f7-920ba8af1e2c.jpg)*  
（运行结果：仅打印 1 到 4）

## continue 语句在 while 循环中的作用

### 功能
- **跳过本次循环的剩余代码**
- **直接回到 while 条件判断处**，开始下一次循环判断

### 示例（易错点！）
```c
int i = 1;
while (i <= 10) {
    if (i == 5) {
        continue;  // 跳过本次循环后续代码
    }
    printf("%d\n", i);
    i++;
}
```

### 问题分析
- 当 `i == 5` 时，执行 `continue`
- **跳过了 `i++`**
- 下次判断时 `i` 仍为 5 → 再次进入 `continue`
- 形成 **死循环**！

### 正确写法（将 `i++` 放在 `continue` 之前）
```c
int i = 1;
while (i <= 10) {
    i++;  // 先自增
    if (i == 6) {  // 对应原 i=5
        continue;
    }
    printf("%d\n", i);
}
```

或更清晰的方式：
```c
int i = 1;
while (i <= 10) {
    if (i == 5) {
        i++;        // 手动更新
        continue;
    }
    printf("%d\n", i);
    i++;
}
```

### 实际运行结果（按视频演示修正后）
- 输出：`2 3 4 6 7 8 9 10 11`
- **没有 5**，且循环正常结束

> 🔍 关键：`continue` 会跳过其后的所有语句，**必须确保循环变量在 `continue` 前已更新**，否则可能死循环。

*![](http://localhost:8000/static/screenshots/screenshot_3e937fe7d-314f-47ab-832f-9a387704789c.jpg)*  
（运行结果：2 到 11，跳过 5）

## break 与 continue 的对比总结

| 特性 | `break` | `continue` |
|------|--------|-----------|
| **作用范围** | 终止整个循环 | 跳过本次循环剩余部分 |
| **执行后行为** | 跳出循环，执行后续代码 | 回到循环条件判断处 |
| **是否影响循环变量** | 不影响（但不再执行） | **极易导致变量未更新 → 死循环** |
| **典型用途** | 满足某条件提前退出 | 跳过特定迭代（如过滤无效数据） |

## 实际应用场景举例

### 场景：字符大小写转换（跳过数字）
```c
char ch;
while ((ch = getchar()) != '\n') {
    if (ch >= '0' && ch <= '9') {
        continue;  // 跳过数字，不处理也不打印
    }
    // 转换大小写并打印
    if (ch >= 'a' && ch <= 'z') {
        ch = ch - 32;
    } else if (ch >= 'A' && ch <= 'Z') {
        ch = ch + 32;
    }
    putchar(ch);
}
```

> ✅ `continue` 在此用于 **跳过不需要处理的数据**，提高代码效率。
